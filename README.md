# HireOS

AI-powered hiring platform that takes candidates from application to a fully scored interview debrief — with zero manual effort from the recruiter.

## What it does

- **Candidate portal** — Candidates register, upload their CV, and complete a 30-minute proctored AI interview from a shareable link
- **AI interviews** — Adaptive questions generated by Llama 3.1 (via Groq), personalised to each candidate's resume
- **Proctored** — Camera mandatory, copy-paste disabled, tab switching flagged
- **AI debrief** — After every interview, recruiters get a full score, verdict, topic-by-topic breakdown, and transcript
- **Pipeline** — Kanban board to move candidates through stages (Applied → Screening → Mock Interview → Founder Round → Technical → Offer)
- **Automated emails** — Invite, advance, and reject emails sent automatically at every stage via Gmail
- **Bulk CV upload** — Upload a CSV of 1,000+ candidates, AI shortlists the top matches in under 2 minutes
- **Multi-tenant** — Each company's data is fully isolated

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS
- **Backend:** Python, FastAPI, SQLAlchemy
- **AI:** Llama 3.1 via Groq API (free tier)
- **Email:** Gmail SMTP
- **Database:** SQLite (local) / PostgreSQL (production)
- **Auth:** JWT (separate recruiter + candidate tokens)

## Local Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

Create a `backend/.env` file:
```
DATABASE_URL=sqlite:///./recruitment_os.db
JWT_SECRET_KEY=your-secret-key
OPENAI_API_KEY=your-groq-key
OPENAI_BASE_URL=https://api.groq.com/openai/v1
OPENAI_MODEL=llama-3.1-8b-instant
GMAIL_USER=your@gmail.com
GMAIL_APP_PASSWORD=your-app-password
EMAIL_FROM=HireOS <your@gmail.com>
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

App runs at `http://localhost:5173`

## Deployment

- **Frontend** → Vercel (set root directory to `frontend`)
- **Backend** → Railway (set root directory to `backend`, add env vars)
